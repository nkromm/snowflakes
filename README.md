# Snowflakes

## Dependencies

```
node: v18.12.1
npm: 8.19.3
```

## Quickstart

To use a random initial state:
```
npm i
npm start
```

or the one in the example:

```
npm i
npm run start:fixed
```

## Development

Use `npm run start:watch` to use `nodemon` to watch the file system and reload the app.
